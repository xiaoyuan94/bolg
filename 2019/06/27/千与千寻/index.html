千与千寻

发表于 2019-06-27 | 分类于 随笔

日本的动漫很多都不是只是给儿童和少年看的东西，很多就是借助动漫的形式来展示我们电影制作而达不到的想象力和奇异的故事背景。

真的是年少看过的东西，和姗姗一起再看，又是另外一番感受。真的是忘了其中的很多部分的情节，再看已经如同一部新的的作品一样去欣赏了。

创作者宫崎骏的的确确可以在日本甚至在整个动漫历史的发展上面留下浓墨重彩的一笔，先不提很多故事的立意深不深刻，光细腻至极的画风，每一帧都如同壁纸一般给人带来的观感，直到现在很多画家都是望尘莫及的。

故事的开篇，千寻的父亲就说他们误入的地方是一个废弃的主题公园，泡沫经济过后这里就被人废弃了，直接就交代了时代的背景。

日本和中国在甲午战争以后，不光是国运逆转，在很多方面他经历彻底的革新之后，对于中国他变成了前行者，很多时候他在发展过程中经历的东西，也变成了中国的前车之鉴。日本战后经济迅速的腾飞，七八十年代，泡沫经济之大，一如中国现在，来到九十年代，日本主动刺破了泡沫，社会也进入了低迷与萧条，而《千与千寻》也正是这样的背景下诞生的一部作品。

经过了鸟居，走过了道祖神，通过了隧道，其实便已经到了神明的地方。

为啥神明的地方，一个个的神明却像是一个个的妖怪一样？这也就是日本神道教的特殊之处，他们有八百万神明，花草树木、江河湖泊、森林鸟兽、人文异想等等，各种各样的事物都可能是神明，所以这也是区别于其他宗教的，独属于日本的一个特色，他们的天皇也是天照大神的化身。

说点题外话，日本的神话很有意思，以前读到伊邪那岐与伊邪那美的故事的时候，描写伊邪那美因为生火神而去世，伊邪那岐如何如何想念伊邪那美，然后远赴冥国黄泉想让伊邪那美救出来，伊邪那美因为吃了冥国的饭食而不能回返，但因为伊邪那岐来找她，她表示愿意跟随他回去，但是要去和黄泉的神商议一下，说罢进了宫殿，伊邪那岐后觉得商议的时间过长便举了火把去宫殿之中，这时看到伊邪那美满身蛆虫蠕动，十分丑陋，伊邪那岐看到这种景象大吃一惊，转身就跑，最后还怕伊邪那美追来，还把黄泉给堵上了，哈哈哈，男人啊。。。。说好的思念呢？哈哈。。

千寻的父母出场不久，就变成了猪的模样，为什么呢？面对无人的街道和香喷喷的食物，确实很多看着就诡异的地方，这里的食物居然敢大张旗鼓的吃起来，最后被诅咒变成了猪，感觉大人不应该都有智慧么？这么容易就陷进去了，但是回过神思考一下，我们现在很多的免费游戏，各种心灵鸡汤的文章，不也就是这样么？例如V社推出的DOTA之类的游戏，例如腾讯之流推出的各种游戏和娱乐产品，确实就是免费游戏和免费的产品啊，但是我们自己还乐呵呵的一遍遍的心甘情愿的享受着免费的游戏，然后被V社和腾讯一遍遍的割韭菜，变成了一个个的猪。另外那些其他的那种大街上就能接触的免费何尝不是这样呢？所谓公知大V，所谓流量明星，所谓免费，等等，他们其实都等着把我们养肥了再去宰杀。。。

印象很深刻的就是小玲和千寻在楼上望着远方的灯火，小玲说出等我有钱了，我一定要去那边镇上看一看。“等我怎么样了，我就如何如何”，我们很多人是不是也是那样，等我。。。可能大多数人的结局都是不可能跳出那个樊笼，或者心中的远方还是待着心中为好，真的到了，可能那份美好就消失了。

无脸男是在网上引起最多讨论的人物，他的角色能确实能引起太多太多的讨论了，迷茫颓废的一代人，不受人重视，空虚幽闭可能那种形象就是类似于无脸男一样，但是进入浮华的地方却又迅速的腐化，变得狰狞可怖， 吐出进入肚里的东西，而后一如既往。可怜的是，千寻应该也不在意他吧，虽然动漫最后给他了一个归宿，可能只是天性使然的友善和礼貌。

所以可能在无脸男被钱婆婆留在那里后，她根本就没有告别，其实根本就不在乎吧。现实生活中有这样的人么？有啊，还真的有啊，更残酷的是现实生活中，很多人没有归宿，最终也没有归宿，属于现实世界，但却游离于现实的透明人。

汤婆婆可能是里面很令人瞩目的智者了吧！

以为哭闹便会获得所有的巨婴，威胁别人的方式也是哭闹，走路都站不稳，不能站立，怀疑外面的世界充满病菌，他跟随千寻转了一圈回来，汤婆婆惊讶的表示你居然会自己站立了。

说起巨婴，其实让我想起一本书《巨婴国》，但是说实话啊，我虽然颂扬批判精深，但是对这种《巨婴国》这种高级喷实在是无语，为了标新立异，真的是刷新下限，别人的爱情是巨婴表现、乱扔垃圾是巨婴表现、希特勒战略失败是巨婴表现、就连太平天国称帝都是巨婴，什么都是巨婴表现。。。。。。无语。

现实中确实存在这种巨婴么？真的也有，日本总是爆出的那种几十年级别的啃老族，几十年不出门，除了一部分是自闭症以外，还真就是真正的巨婴啊，中国这种情况新闻上也屡见不鲜。

记得韩国有一部电影《金氏漂流记》，里面女主角就是几十年不出门的人，她是那种有自闭症的人，男猪脚是个因为生活不如意，自杀但是却意外被困在荒岛的人，哈哈哈，故事是真的魔幻的可以，一个在岛上画沙，一个用漂流瓶进行交流，最后却在荒芜的人生中找到了亮光，最后走出了封闭她的大门。希望那些因为病症的人可以找到光亮走出那个心理和不足几米的大门，也像《千与千寻》的宝宝一样，独自站立起来。


其实里面的主要出场角色我感觉都不是我们，其实小玲和一众工作的女人和蛙人才是一个个我们，崇敬金子和浮华，但是又真心欢呼千寻胜利解救父母。

在楼地下烧水的老爷爷也拥有颗滚烫的心。

一个个头顶着器皿等待着无脸男发金子的大众，也会手拿着一颗香烟，站在窗前不知道是不是也会想起生活的辛苦和家人的期待，其实他们确实更像一个个生活在现实的那些我们，平凡但及其真实的活着。

摇撸帮助千寻去找钱婆婆的小玲

千与千寻也是个关于小女孩成长的片子，从一开始抱怨父母，不会体谅别人，都最后经历了各种各样的经历，然后最后也成长了起来。

是啊！很多时候我们人生中很多的人和事都会慢慢的离我们越来越远，甚至以后可能再也不见了。但是向前了就不要回头了。

江湖路远，道声珍重！

很多时候随着我们成长，父母也不会再牵着你的手，你想留恋但是也不能，也不可能回头了。可能那头上汤婆婆的头绳就是我们经历过所有的留念。所以我们在和他们有缘分的时候一定珍惜啊，那份可能再也回不到的时光。

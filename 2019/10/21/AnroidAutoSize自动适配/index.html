AnroidAutoSize自动适配

发表于 2019-10-21 | 分类于 开发

简介：

众所周知Android的机型多种多样，各种分辨率又琳琅满目，所以造成的问题就是APP在适配的时候就会及其麻烦，一直以来都要对所有的分辨率都要弄一种单独的配置，极度麻烦。

现在针对此问题，头条开源了一个适配方案，原理就是在不同分辨率下，自动修改当前控件的布局，让所有界面的观感都几乎一样。

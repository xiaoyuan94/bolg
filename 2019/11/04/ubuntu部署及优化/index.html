Ubuntu安装

具体安装不再赘述，简要说几个安装当中需要注意的几个点。

分区

分区是安装中需要特别注意的点,当然你可以直接使用默认的安装模式来安装,但是可能你不知道里面的细节而导致一些问题的出现.

主要需要分这么几个区:

1,挂载/ 的根目录,这个目录存放的是Linux系统的的主要文件,一般分超过30g即可,具体按照自身的硬盘的大小来分,可以大,但是不能过小,防止升级系统出现硬盘容量不够的情况.

2挂载 / boot的启动目录,1g即可

3,挂载 swap内存交换空间,具体和自身的内存总容量保持一致.

4,挂载/home的用户目录,其他剩余空间都可以分配给这个.

另外注意下,启动的efi,若是双系统使用的用户,一定注意下方的启动目录使用efi所在的目录,不然引导会出现不成功,出现无法启动的情况

安装

开始安装不用使用wifi连接,因为在安装过程中回去网络下载很多的资源,减慢安装部署速度,开始可以尽量离线安装即可,后续在修改国内比较快的镜像来更新即可.

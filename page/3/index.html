心学FAQ

发表于 2018-12-10 | 分类于 随笔

心学的东西，每看一次都有一次的收获。

最近看了一个帖子，有人说：我心里很讨厌一个人，这个时候我要不要表现出来呢？

阳明先生对于这种事情说的很清楚：身之主宰为心，心之所发为意，意之本体是知，意之所在便是物。所以说讨厌这个人，很多时候在不了解这个人的时候，初次看到就发觉讨厌，但是这个讨厌是刻板印象，这个印象来源于哪里呢？其实是对某些自己厌恶的东西延伸而来的，讨厌这个人，其实是讨厌这个人可能出现的品行或者其他厌恶的行为，所以该不该表现出来呢？讨厌是一种意的表现，若是确实有了应该讨厌的行为，那就表现出讨厌这中行为，这个行为是否应该讨厌？评判标准就是知，关键点还是知。

但是现在感觉是自己没有活明白，读的书还是太少，其实是被社会同化了，世故、圆滑，平庸、无趣、而且恶心。

简单美好其实就是更容易亲近正能量的东西，其实那个才是生活本质，才是活着的意义。其实更可贵啊！其实是自己迷失了，一直都觉得是那样是非常美好的，但是一直说不清楚美好在哪里。

看山是山，看山不是山，看山又是山！

社会还真的这样啊！！！大人大多没有小孩子素质高的年代，是个小孩子要拉着大人说要遵守交通规则的年代。

但是最好其实我们自己知道其实最初的坚持其实就是对的！！！

立 志

志不立，天下无可成之事。虽百工技艺，未有不本于志者。今学者旷废隳惰，玩岁愒时，而百无所成，皆由于志之未立耳。故立志而圣，则圣矣；立志而贤，则贤矣；志不立，如无舵之舟，无衔之马，漂荡奔逸，终亦何所底乎？昔人有言：「使为善而父母怒之，兄弟怨之，宗族乡党贱恶之，如此而不为善，可也。为善则父母爱之，兄弟悦之，宗族乡党敬信之，何苦而不为善、为君子？使为恶而父母爱之，兄弟悦之，宗族乡党敬信之，如此而为恶，可也。为恶则父母怒之，兄弟怨之，宗族乡党贱恶之，何苦必为恶、为小人？」诸生念此，亦可以知所立志矣。

梁启超先生说佛之业与轮回

发表于 2018-10-20 | 分类于 随笔

梁启超是自己读过以学者角度来讲佛的，个人觉得应该是最深入人心的，区别于圣严法师的用简单和朴素的语言来讲述到底什么才是正信的佛教？到底学佛和佛学是在干什么？破处一般认知中迷信的认识，了解佛学，自立自信自强。而梁启超先生不讲经不讲论，更不讲律，他用自己学者的思维来说他研究的佛学到底是什么东西。他们也区别于一些写心灵鸡汤的所谓佛学的“高僧”，实在是无甚意义。

上学的时候读《少年中国说》，读完的感受真的难以形容。“少年智则国智，少年富则国富；少年强则国强，少年独立则国独立；少年自由则国自由；少年进步则国进步；少年胜于欧洲，则国胜于欧洲；少年雄于地球，则国雄于地球。红日初升，其道大光。河出伏流，一泻汪洋。潜龙腾渊，鳞爪飞扬。乳虎啸谷，百兽震惶。鹰隼试翼，风尘翕张。奇花初胎，矞矞皇皇。干将发硎，有作其芒。天戴其苍，地履其黄。纵有千古，横有八荒。前途似海，来日方长。美哉我少年中国，与天不老！壮哉我中国少年，与国无疆！”振聋发聩一般。

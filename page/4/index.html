教条示龙场诸生

发表于 2018-10-29 | 分类于 随笔

诸生相从，于此甚盛。恐无能为助也，以四事相规，聊以答诸生之意。一曰立志，二曰勤学，三曰改过，四曰责善。其慎听毋忽！

立 志

志不立，天下无可成之事。虽百工技艺，未有不本于志者。今学者旷废隳惰，玩岁愒时，而百无所成，皆由于志之未立耳。故立志而圣，则圣矣；立志而贤，则贤矣；志不立，如无舵之舟，无衔之马，漂荡奔逸，终亦何所底乎？昔人有言：「使为善而父母怒之，兄弟怨之，宗族乡党贱恶之，如此而不为善，可也。为善则父母爱之，兄弟悦之，宗族乡党敬信之，何苦而不为善、为君子？使为恶而父母爱之，兄弟悦之，宗族乡党敬信之，如此而为恶，可也。为恶则父母怒之，兄弟怨之，宗族乡党贱恶之，何苦必为恶、为小人？」诸生念此，亦可以知所立志矣。

重构Jsbrige项目

发表于 2018-09-29 | 分类于 开发

Jsbrige是一个很好的原生接入H5的好的方案，轻量便捷，易于上线，但是不好的地方就是性能比这其他的类似RN，或者WEEX的方案要差不少，但是会引起的问题也会少很多，那种用js来写，底层驱动还是原生的VIEW的方案，遇到问题的时候要搞死人。

RN和WEEX这种大公司的框架非常好，但是他们是耗费了大量的人力和物力来去搞，有大量的人来解决和维护这个，但是对于独立开发或者需要快速上线的，实在是成本太高了。

总而言之，Jsbrige是个很不错的方案，而且现在流行的VUE,react，anglar这种框架，或者用ES、coffeescript、TS这种新式JS，都可以直接build以后来用，甚至是写好的web端都可以直接重构适配一下，直接就可以用。

再见深圳

发表于 2018-08-06 | 分类于 随笔

又一次背上行囊。

这里是个很不错的城市，喜欢这种一年四季温差都不大，空气质量极佳，常年几乎都可以穿个拖鞋到处晃悠的地方，一线都去过，应该没有比这里感觉更好的地方，对了唯一感觉不适应的就是太潮了。

打包的时候才发现，很简单生活的我，居然也会有这么多的零零散散的东西，整理一下，居然也需要好几个包。

一直在用的神船，用来做工作偶尔玩个游戏，看个高清都可以，同价位性价比应该没有比这个更高了的吧，被人诟病小毛病多，也没有出现。

去年已经寄回家一箱子了，居然书还需要装两箱。最黑暗的日子，小陈远来住了很长时间，除了朋友，也就是自己的吃饭的家伙和这些东西陪着自己挺了过去。

弹了两个月就被我束之高阁，中间也就陈远来住的时候拿出来划拉了两下，然后又被我弄去睡大觉了。自己安排了太多的学习的任务，时到今日还有很多没学完的，其实很多碎片的时间被自己浪费了，应该用来多练练。

来这里算是很有收获，开阔了视野，学了很多很多的知识，也结识了很多朋友，身边的人也都很好，领导不错，分内的事情搞完，时常的懒散，也从来没有说过不是，同事朋友也很好，解释了太多的工作中疑问，吊哥帮自己找的房子住，翔翔发短信问过：家里是不是有什么事情？需不需要帮忙？小斌斌好不容易解决了自己的个人问题后，中间居然还要给我介绍他媳妇的朋友，晃晃也要介绍他觉得不错的女同学，伟忠天天被小陈远吐槽的色咪咪的小眼神，一个个的人，回想一下其实很怀念，也很感动，就连住了两年多的房子的房东太太都没有给我涨过价。

已经失去热情了，那就88喽，换个地方，离家近一些，钱少挣一点，过的像更像生活一些，以前刻意忽视的东西，却不知那才是生活的全部。

让《一人之下》这部漫画升华的一段剧情–陈朵篇。 当看到陈朵最后要和众人上街度过最后的时光的时候，那个场景的旁白让我想到了另外的一部神作《新世纪福音战士》，其实两个漫画没有任何关联的地方，纯粹是自己的联想而已。 好伤感的故事啊，在其中的一段剧情中介绍陈朵的一生，短短的人生啊，从旁人的眼光中是怎样的一种苦难，但是特殊的经历，陈朵可能不知道那个是苦难，被人当做杀人工具和蛊虫的容器，特殊的训练把她变成了武器，虽然无时无刻都要经历着非人的苦难，但摒弃了人正常的感受，让她不知道自己这个是苦难，后来组织中的那些人怀着善良的心态，捣毁了那个邪恶的药仙会，从组织的立场上看似拯救了她，也改造了她，她也觉醒了人性，但是后果就是她知道了自身的苦难，反而自身先承受不了这种东西，她也知道她永远都无法融入正常人的生活了，于是就有后来张楚岚给予她一个死亡的选择，她也接受了这样的结果。

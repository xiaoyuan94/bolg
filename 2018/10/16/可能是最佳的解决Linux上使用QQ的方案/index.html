可能是最佳的解决Linux上使用QQ的方案

发表于 2018-10-16 | 分类于 开发

开篇说点题外话，各个系统用来用去，最后我发现你没有理由不用ubuntu针对linux的各个版本，还方便学习后台开发、大数据开发、分布式部署、嵌入式开发，可能是最适合、最中庸的发行版。

完全不懂使用的话，建议还是用deepin，国产没有任何理由来歧视，比所谓的国外的很多发行版做的要吊的多，但若是平常工作中不接触linux的话，还是使用win10吧，简直是艺术一样的桌面系统，人家搞得这么吊，你没有你理由不用不是么？

最后来说下其他的简单的常见的发行版

mint:其实还是ubuntu。

manjaro :比它的原生发行版要稳定一些，但是还是有arch的毛病，滚动更新既是优点，但同时也是缺点，一不小心就会滚动挂掉的东西，还是放弃吧，系统是拿来用的，不是折腾的，另外用来学习没有任何优势，服务器应该没有那这个来使用的，随时都能滚动挂的东西。。。。manjaro要好一点，相对稳定很多，可以考虑使用。

fedora：优点就是稳定，和它父亲一样红帽的东西就是这个特色，坚若磐石，所以说软件就。。。yum的源你懂得,有的太旧了,需要自己动手的有点多。

总之最后我还是选择了ubuntu，庞大的社区，太多人用了，至于什么排名什么的，仁者见仁吧，若是你知道黑客的系统定制版就是用的ubuntu14版本，单论linux的用户群体它毫无疑问是最多的，不好的地方就是现在官方把精力更多的放在了ubuntu server上面，因为移动端的ubuntu的失败，所以关注度下降了很多，但是还推荐使用，适当的折腾就可以很好的来用这个系统，apt install 已经可以解决大部分问题了。

安装qq等等window上面常用的软件

1，安装deepin-wine环境：上https://gitee.com/wszqkzqk/deepin-wine-for-ubuntu页面下载zip包（或用git方式克隆），解压到本地文件夹，在文件夹中打开终端，输入sudo sh ./install.sh一键安装。

2，安装deepin.com应用容器：在http://mirrors.aliyun.com/deepin/pool/non-free/d/中下载想要的容器，点击deb安装即可。以下为推荐容器:

QQ：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.im/

TIM：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.office/

QQ轻聊版：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.qq.im.light/

微信：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.wechat/

Foxmail：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.foxmail/

百度网盘：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.com.baidu.pan/

360压缩：http://mirrors.aliyun.com/deepin/pool/non-free/d/deepin.cn.360.yasuo/

3，Ubuntu 18.04 Gnome桌面显示传统托盘图标：安装TopIconPlus的gnome-shell扩展，命令：sudo apt-get install gnome-shell-extension-top-icons-plus gnome-tweaks，然后用gnome-tweaks开启这个扩展。

插件需要重启一下来使用。

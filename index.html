一统江山之康德

发表于 2020-01-07 | 分类于 随笔

世界上唯有两样东西能让我们的内心受到深深的震撼，一是我们头顶浩瀚灿烂的星空，一是我们心中崇高的道德法则。——康德

光看人生的历程，康德的人生历程真的是乏味，没有什么大风大浪的过程，更谈不上腥风血雨，做的最多的事情可能就是老老实实的教学了，甚至婚姻生活的都没有，大哲学家的历程可能就是那么的独特，看似乏味的人生一点也不能遮挡他的伟大。

康德是德国哲学革命的旗手和德国古典哲学的创始人。康德以他的批判性哲学为基础，提出了人是目的而非手段，实现了将人从单纯因果必然性的束缚下解放出来，完成古典法哲学向现代法哲学转变和发展史上革命性变革。同时，康德对人的自由和权利的实现和维护、以及私法和公法等法哲学问题进行了深入的探讨，形成了丰富的法哲学思想，对于后代法哲学的发展产生了重大的影响。

康德的批判哲学体系是近代理性主义和经验主义矛盾的结合，具有鲜明的二元论特征。他的哲学对德国古典哲学和西方哲学具有深远影响，对马克思主义哲学的诞生也具有深刻影响

浅谈三我

发表于 2020-01-02 | 分类于 随笔

本我、自我与超我是由精神分析学家弗洛伊德之结构理论所提出，精神的三大部分。

弗洛伊德提出相关概念，以解释意识和潜意识的形成和相互关系。

“本我”（完全潜意识）代表欲望，受意识遏抑；

“自我”（大部分有意识）负责处理现实世界的事情；

“超我”（部分有意识）是良知或内在的道德判断。

在弗洛伊德的理论中，意识所分为的三部分，即本我，自我，超我构成了人的完整的人格。人的一切心理活动都可以从他们之间的联系中得到合理的解释，自我是永久存在的，而超我和本我又几乎是永久对立的，为了协调本我和超我之间的矛盾，自我需要进行调节。若个人承受的来自本我、超我和外界压力过大而产生焦虑时，自我就会帮助启动防御机制。防御机制有：压抑、否认、退行、抵消、投射、升华等等。

ubuntu部署及优化

发表于 2019-11-04 | 分类于 开发

Ubuntu安装

具体安装不再赘述，简要说几个安装当中需要注意的几个点。

分区

分区是安装中需要特别注意的点,当然你可以直接使用默认的安装模式来安装,但是可能你不知道里面的细节而导致一些问题的出现.

主要需要分这么几个区:

1,挂载/ 的根目录,这个目录存放的是Linux系统的的主要文件,一般分超过30g即可,具体按照自身的硬盘的大小来分,可以大,但是不能过小,防止升级系统出现硬盘容量不够的情况.

2挂载 / boot的启动目录,1g即可

3,挂载 swap内存交换空间,具体和自身的内存总容量保持一致.

4,挂载/home的用户目录,其他剩余空间都可以分配给这个.

另外注意下,启动的efi,若是双系统使用的用户,一定注意下方的启动目录使用efi所在的目录,不然引导会出现不成功,出现无法启动的情况

安装

开始安装不用使用wifi连接,因为在安装过程中回去网络下载很多的资源,减慢安装部署速度,开始可以尽量离线安装即可,后续在修改国内比较快的镜像来更新即可.

odoo简介

选择odoo理由

应该是最好的一款开源的ERP软件了吧，几乎没有之一。

涵盖了你能想到的正常中型公司几乎所有涉及到的业务线，社区版本就能满足需要了，若是想要一些定制型的开发，就要收点费用了，当然也无可厚非，但是呢，强大就强大在于没有什么用户数量，或者什么权限之类的限制，源码也是开源的，懂python开发的，自己也可以做定制型的开发，确实应该是开源类的ERP中最良心的了，没的说，其他的开源要么没做完，要么就是个半吊子，再要么就是做的东西实在是烂。

总之，odoo值得选择。

另外虽然是标榜的ERP软件，但是对于正常电商业务的支撑一点不差，能有的几乎都有，一站式将公司从小做到中型公司，真的很不错。

什么是odoo

Odoo 是一个开源框架，针对 ERP 的需求发展而来，适合定制出符合客户各种需求的 ERP 系统和电子商务系统。

由于 Odoo 的底层结构很强大，所有模块都可以根据需要进行安装和御载，所以客户能像搭积木一样建构自己期望的系统。即使完全不安装与ERP相关的模块，仅利用 Odoo 自身的底层框架，也能构建出适合各种用途的系统，纯 B/S 架构。而这一切都是免费的，并且不受限于源代码，也不受于用户数。

Odoo 的主要模块包括：销售、采购、库存、财务、生产、项目管理、CRM、POS、人力资源、车辆管理、午餐管理等等。

三年学话，一生闭嘴

发表于 2021-01-01 | 分类于 随笔

郭老板尝尝挂在嘴边的一句话，三年学说话，一生学闭嘴，真的年纪越大越体会其中滋味。

写在2021的第一天，人受情绪的掌控越深，越不容易控制自己，很多不经思索的话就从嘴里出来了，三思再说可能也是错的，不如一默，好好做事，不用去解释太多。

想起好多年前有一次和同学吃饭，同学说：你的思想太偏激了。转过头来让自己思考了很久很久，不知从什么时候自己确实的思维确实有点偏激，今天去看了华仔的新片拆弹专家2里面饰演的人就是经历事情后从开始一个个偏激的思维，慢慢变成了一个仇恨社会的恐怖分子，虽然里面人物性格转变有点生硬，最后的情节处理也不算很好，但是总体上算是个不错的片子，偏激思维到仇恨社会到不至于，偏激的思维确实不好，现在慢慢有意识的回复别人的时候让自己的想法中庸，但是还是偶尔控制不了自己从嘴巴中说出来的事情，思索之后再说其实很多也不恰当，总结以后感觉不如一默，好好的做事就好，不要让情绪引导自己，多多用这些时间去思考，多去做事。

回想起来偏激的思维是从什么时候开始呢？什么原因呢？觉得还是因为自己做事情的方式导致的，自己为了让事情处理简单起来，下意识会让事情分类，然后一刀切，这种处理方式好处是能够快速的解决问题，但是随之而来的问题就是处理事情考虑是不够周全的，而且若是选择处理方式是错误的话，无法做出快速修正，因为一刀切完全走向了错的方向，方向会越来越偏离，越来越错，什么时候开始的呢？可能是因为妈妈走了以后，自己生活也受了太多苦，很多时候在负能量中就是慢慢的走向偏激，以后不能这样，控制自己，战胜自己，才是真正的强大。

现在自己的思维和心理，自己反思审视一下自己已经无意识中已经出问题了，黑暗和深渊离自己很近，需要警惕之，慎行之，笃行之，另外有些傲气带来的沾沾自喜也要去掉。怎么解决？

1，不断反思。用第三人的视角去审视自己，修正自己，慢慢变成习惯，优化后让自己增加正能量。

2，每天用一定时间去冥想。冥想能够慢慢增强心灵的力量，心灵强大才能支撑自己的不被意外和变化影响自己。

3，处理冥想及现实生活中的杂乱的念头。专注冥想中，会产生很多很多杂乱念头，这种东西不是想没有就没有，怎么办？若是无法摒弃，放开心灵让念头都来冲刷自己，但守着一个根本念头，让其他念头来不断冲刷，增强意志力，慢慢让其他念头消失。

4，增加执行力。强迫自身去做事情是非常困难的，比如简简单单的早起就很痛苦，甚至是人会给自己找很多理由让其不断拖延，这个真的很难。要用时间来控制，每日什么时间做什么事情一定要心中有数，若都无法坚持，先从每日要练习写代码开始，最近看了一个电视剧《棋魂》，想要有所成就，其中围棋九段俞晓阳每日都要练习，对于个人一样，所有的事情都可以拖延包括工作、生活，但是为之一生的事业不行，需要长期持续不断的努力才行，一点点，一天天。

5，闭嘴。心中无名火起，偶然的炫耀心，谈论杂事，思考一下现在的思考，若是通过语言无法解决的话，就闭嘴，若是没思考好，就先闭嘴，若还无法做到大智若愚，就先闭嘴，毛主席说过不谈有三：不谈钱、不谈男女之爱、不谈琐事，记住谈论的事情是这些的话那就闭嘴。

人生第一次被关局子，被警察关到号子里到半夜的时候，多么希望想要过一生人能哪怕关心一下、想起我一下，被老爹捞出来，躺床上盯着屋里天花板居然哭了好久，稍不如意连送送我都不愿意去了，晚上坐在车上，盯着上铺，又哭了好久，真的好懦弱啊，2020庚子年，给自己的伤害太多了，所幸自己经历过更痛苦的时段，虽然厌恶自己的懦弱，但是已经打不倒自己，爬起来，站起来还是要继续出发。

2021新的征程，年尾各种压力快要喘不上气的时候，偶然的机会换了个工作进入另外一个层次，要在新的一年正式开启新的篇章，希望努力后能够大有收获，收获多年之前和同学吃饭时笑谈起的梦想，照进现实就在前方，加油。

祖龙之谜

发表于 2020-12-14 | 分类于 随笔

最近在看热播的大秦赋，真的是期望越大失望是有多大。

始皇的可以查的历史很多，基本一生关键事件都有详细记载，但是后世抓着黑的人也不在少数，汉朝贾谊的过秦论想必读书的时候几乎每个人都有印象，给我们的印象就是秦残暴，所以才有二世而亡。

但是真正读了很多书以后，项羽这个在易安居士诗中至今思项羽，不肯过江东的霸王，比之残暴过之而无不及也，屠城，杀降，抢劫那一项都是干的让人发指，更像是个残暴的强盗，另外动荡的楚汉争霸死的人，到汉景帝还没有缓过劲，这就是所谓的可奈何的项羽。

说秦国残暴大多说秦政苛法不仁，天下苦秦久矣。可是呢，汉承秦制几乎就是略微改进而已，秦税十税一，楚汉之间死的人太多了，到了汉文帝景帝也没缓过劲来，在加上景帝初清君侧的八王之乱，民生实在是凋敝，才有了文景之治的十二税一，就是汉武极盛之时，比之秦帝国的国力也两可之间，当然并非为秦法彻底洗白，秦法里面有很多确实是挺苛的，各种徭役啊，连坐啊，各种肉刑啊，但是封建王朝时代那个王朝其实都好不到哪去，历史局限性，但总体上是没有大的问题。

陈胜吴广起义声势浩大，好像官逼民反一样，其实呢，还是六国那些贵族不甘心被剥夺权利，高举反旗，都来投机分一杯羹，说白了还是始皇帝立国之本为法制，没有彻底将之清算干净，又骤然驾崩，再加上内部的种种的黑天鹅事件一个接着一个，能征善战的老了，忠心不二的内侍近臣，治理国家的能臣都随着始皇的突然死去变得野心勃勃，帝国最为关键的权利交接没有做好，造成了后来之局面，越来越糟。

hexo博客的搭建

发表于 2020-01-13 | 分类于 开发

前言

突然发现没有写过自己的博客是怎么弄的？关键字：hexo

这个博客已经运行一年半了，之前一直用的是php+WordPress+mysql的架构，用阿里的服务器运行了两年，遗憾的是之前的博客数据因为架构不一样，不能导入新的博客当中，很多之前写的都gg了。

之前的方案是完整的网站的运行方式，但对于我这种只是运行一个博客太浪费了，另外还要自己维护服务器+网站+数据库，太浪费时间了，转到这种hexo的方式，简单好用，只用申请一个域名即可，其他的全都是用现成的git的page来托管的，不用数据库，服务器是用别人的托管的不用自己维护，防火墙安全工作别人都帮你解决了，国外用github托管、国内使用coding托管，国内外都能快速访问，对于个人的使用者再好不过。

废话说多了，言归正传！

架构

hexo + next主题+node+coding+github+markdown

架构中其实只有一个关键hexo这个框架，非常简单实用的一个博客框架。

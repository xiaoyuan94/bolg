《惊悚乐园》--永远的Joker

发表于 2017-06-12 | 分类于 随笔

忘了从什么时候接触了这本小说，开始看的需求其实是为了看的爽，开始看的时候差点就弃书了，但是往下看的时候，却发现写的真的不错。

吐槽、幽默、腹黑、中二的主角，区别于其他小白文中的设定，无限流的故事框架，但是又写出了不一样的感觉，有些故事的脑洞之大让人叹为观止，各种各样的神展开，秀三观，有不少故意秀的成分，总体上还算不上文学著作，但是却让人印象深刻。

比较佩服作者的各种各样梗的来源之广，侧面也反应了作者阅读量确实是不少了，对于大量推理小说、漫画、电影、文学名著，另外也像福尔摩斯一样对各种各样的知识也都要涉猎，也都有自己的研究和观点，也能在各种各样的神展开中能移植到自己的情节里面，读起来发现这些梗的时候，还能启发自己去寻找对应的东西去了解一下，发现不一样的世界。

介绍最近的一个短篇情节—-说出我的名字。

里面介绍‘英雄’这两个字真正的含义和精神让人印象深刻，也反映了作者的一种人生观，看似疯癫的嬉笑怒骂，但其实三观并非不正，主角里面自嘲自己反人类，反社会，自恋，宅，但其实呢，更多是的是对社会很多病态的吐槽。他其实就是小丑，自己理解的小丑，但捍卫蝙蝠侠所代表的正义，只打击犯罪，不杀人。

蝙蝠侠就像中国历史上的墨子一样，当世时，世人皆不行仁义，但墨子仍然坚持仁义，并不因世人不行仁义而放弃仁义，而自己的学说也慢慢被大多数人接受，成为与儒学并列的显学，这就是英雄。

DC宇宙区别于漫威宇宙，虽然都是表现超级英雄的漫画，但是DC宇宙却又有来自它的更深的讨论，对于社会、人性、独特的哲学。

记得刚看完《黑暗骑士》的时候，那个时候才知道超级英雄的片子居然也可以这么的让人深思，应该作为配角的小丑却吸引我的全部目光，正义与邪恶我觉得都不能彻底形容这个角色，另外这个演员的表现也让这样的角色塑造变成的绝唱，伴随癫狂独特的笑声，一切的一切不过是他的一场游戏，他喜欢混乱，极度的混乱才能实现真正的公平，希望自发的由有序向无序发展，他愿意看到人们之间的争斗，这种公平恰恰是法制所做不到的，而法制所追求的，真的就是公平吗？

“Why so serious!”小丑洞悉了人性中的恐惧，也利用了这些恐惧和弱点，人们所颂扬光明骑士Harvey也被成功腐蚀为双面人，而黑暗骑士是Joker还是蝙蝠侠呢？他看透了荒诞的生活，蝙蝠侠忍受了这种荒诞，甚至是保护了那些普普通通的大众，而Joker是不能忍受的，他清醒所以难以忍受，于是他就要把荒诞毁灭，带来虚无与混乱。
